# ModsenTask

This project is a test task for Modsen

## Technologies

1. Spring, Boot.
2. ORM:Spring Data Jpa.
3. RDBMS: PostgreSQL.
4. Kafka;
5. Flyway;
6. Swagger
7. JUnit

## Installation

- Java 21
- PostgreSQL
- Docker

## Starting

1. Clone this repository
```
https://github.com/konst-mukhin/ModsenTask.git
```

2. Go to "library" directory
```
mvn clean package
```

3. Run docker
```
docker-compose up -d
```
