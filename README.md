# "ModsenTsk"
## Description
This project is a test task for Modsen.
## Technologies
- Spring, Boot
- ORM:Spring Data Jpa
- RDBMS: PostgreSQL
- ModelMapper
- Kafka
- Swagger
- JUnit
## Installation
### Install the following
- Java 21
- PostgreSQL
- Docker Desktop
### Docker
Clone repository: 
```
git clone https://github.com/konst-mukhin/ModsenTask.git
```
Build and run:
```
docker-compose up -d
```
